# meurepositorio
Aprendendo a configurar e usar o GitHub, uma plataforma de hospedagem de código e controle de versão.
O GitHub é amplamente utilizado para colaboração em projetos de desenvolvimento de software. Você
aprenderá como configurar sua conta no GitHub, criar um repositório, clonar um repositório existente, fazer
alterações, adicionar e confirmar as alterações, e enviar as alterações de volta ao repositório remoto.
